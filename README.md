This repository contains an Enterprise Programming practical project using **Hibernate ORM with Maven (POM configuration).**

---

## 📌 Project Name

ep-expriment-hibernate-POM  
Folder: amithibernate01

---

## 🎯 Objective

To implement database operations using **Hibernate Framework** with **Maven dependency management**.

This project demonstrates:

- Hibernate configuration
- Entity class mapping
- Database CRUD operations
- Maven project structure
